# ⚽🏆 GOLSYSTEM

Sistema profesional de gestión de torneos de fútbol amateur y semi-profesional.

Permite administrar torneos completos desde su configuración hasta su finalización, incluyendo equipos, jugadores, fixtures, resultados, estadísticas y control disciplinario.

---

## 🚀 Características

✔ Creación y configuración de torneos  
✔ Gestión de equipos y jugadores  
✔ Generación de fixture (calendario)  
✔ Soporte para:
- Liga (todos contra todos)
- Fase de grupos
- Eliminación directa  

✔ Registro de resultados (tiempo regular, extra, penales)  
✔ Tabla de posiciones automática  
✔ Sistema disciplinario (tarjetas y suspensiones)  
✔ Estadísticas (goleadores, rendimiento)  
✔ Historial de campeones  

---

## 🏗️ Arquitectura

El proyecto sigue el patrón:

**MVC (Model - View - Controller)**

- **Model** → Entidades JPA (Base de datos)
- **Repository** → Acceso a datos
- **Service** → Lógica de negocio
- **Controller** → API REST
